# PlantMind AI

AI-powered document assistant built using React and FastAPI.

## Features

- Upload PDF, DOCX, PPTX, XLSX, CSV, TXT
- AI Chat based on uploaded document
- Document Search
- Automatic Summary
- FastAPI Backend
- React Frontend

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS

Backend
- FastAPI
- Python

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
